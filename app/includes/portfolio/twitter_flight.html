Twitter Flight Developer Conference 2015

October 2015 UI/UX, Design, Development

Project Description

For Twitter Flight 2015, Twitter wanted to show off real-time data visualizations using their APIs in an interactive art installation format. Six talented developers and data scientists created unique visualizations that also drove physical installations, and I designed and built a navigation interface that allowed visitors to interact with and learn more about each visualization. The navigation UI was displayed on a 50" touchscreen, with the visualization mirrored onto a wall of 6 large displays. Additionally, there was an iPad next to the touchscreen that acted as an art gallery placard where visitors could read further details about each visualization. There were three of these setups at the conference.

Technologies Used

Adobe Illustrator, HTML, CSS, Javascript
