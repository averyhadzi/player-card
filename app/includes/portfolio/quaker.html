Project Description

BlogFrog was one of multiple agencies to contribute to the Address Your Heart campaign/contest. We designed and developed a widget ecosystem that tied into the main microsite. Bloggers embedded our conversation widget and sidebar units, where readers could directly submit an entry for the contest. The contest entries would then appear in the main widget on the microsite, where readers could then vote for all of the entries.

Technologies Used

Adobe Photoshop, Adobe Illustrator, BlogFrog Platform, HTML, CSS, Javascript
